# 🎤 Voice-Based Attendance System

An AI-powered Voice-Based Attendance System built using **Python**, **Flask**, and **SpeechBrain**. The application authenticates users using their voice and automatically records attendance through a simple web interface.

---

## 📖 Project Overview

Traditional attendance systems require manual intervention or physical interaction. This project uses **speaker recognition** to identify registered users from their voice and automatically mark attendance, providing a faster and more secure process.

---

## ✨ Features

- 🎙️ Voice-based user authentication
- 👤 User enrollment
- ✅ Automatic attendance marking
- 🌐 Flask web interface
- 🧠 AI-powered speaker recognition
- 📊 Dashboard for attendance

---

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/chowdaryliyakath/Voice-Based-Attendance-System.git
```

### Go to the project folder

```bash
cd Voice-Based-Attendance-System
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```
